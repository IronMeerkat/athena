# Features

Current capabilities and components:

- Deterministic policy engine (foundation): schedules, allow/deny lists, strict mode
- Background orchestrator (MV3 service worker) with typed messaging
- Content capture and UI overlays (content, content-ui, content-runtime)
- Options page with Zustand store for settings
- Popup, new-tab, and side-panel React apps
- Shared UI components and Tailwind theming
- i18n infrastructure and shared locales
- LLM provider abstraction ready for classification/appeal (OpenAI adapter scaffolded)
- Zip packaging and HMR tooling for fast dev
